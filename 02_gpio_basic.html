第二章 GPIO基础与应用

1. GPIO概念

什么是GPIO？

GPIO（General Purpose Input Output）即通用输入输出端口，是MCU与外部世界交互最基本的接口。

主要用途：

  • 控制外部设备：LED、继电器、蜂鸣器等
  • 读取外部信号：按键、传感器数字输出等
  • 通信协议模拟：软件模拟I2C、SPI等

2. GPIO工作模式与原理

STM32F103的GPIO共有8种工作模式：

输入模式（4种）：

  • 浮空输入
  • 上拉输入
  • 下拉输入
  • 模拟输入

输出模式（4种）：

  • 推挽输出
  • 开漏输出
  • 复用推挽输出
  • 复用开漏输出

关键参数：

  • 输出速度：低速、中速、高速
  • 驱动能力：推挽可输出高低电平，开漏需外接上拉

3. GPIO使用方法

3.1 CubeMX配置步骤：

  1. 在引脚图上点击目标引脚，选择GPIO模式
  2. 配置GPIO属性（上下拉、速度、标签等）
  3. 生成代码

3.2 HAL库常用函数：

// 写引脚电平
HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);

// 读引脚电平
GPIO_PinState state = HAL_GPIO_ReadPin(GPIOA, GPIO_PIN_0);

// 翻转引脚电平
HAL_GPIO_TogglePin(GPIOA, GPIO_PIN_5);
